# <span data-i18n="docsPage.gettingStarted.title">Getting Started</span>

<span data-i18n="docsPage.gettingStarted.intro">This guide walks you through adding the HelmForge repository and installing your first chart.</span>

## <span data-i18n="docsPage.gettingStarted.prerequisites">Prerequisites</span>

<span data-i18n="docsPage.gettingStarted.before">Before you begin, make sure you have:</span>

- **Helm 3.x** <span data-i18n="docsPage.gettingStarted.helmInstalled">installed</span> ([<span data-i18n="docsPage.gettingStarted.installGuide">installation guide</span>](https://helm.sh/docs/intro/install/))
- **kubectl** <span data-i18n="docsPage.gettingStarted.kubectlConfigured">configured with access to a Kubernetes cluster</span>
- <span data-i18n="docsPage.gettingStarted.cluster">A running</span> **Kubernetes cluster** (<span data-i18n="docsPage.gettingStarted.clusterVersion">v1.24 or later recommended</span>)

## <span data-i18n="docsPage.gettingStarted.addRepo">Add the Repository</span>

### <span data-i18n="docsPage.gettingStarted.https">HTTPS Repository</span>

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
```

<span data-i18n="docsPage.gettingStarted.verify">Verify the repository was added:</span>

```bash
helm search repo helmforge
```

### <span data-i18n="docsPage.gettingStarted.oci">OCI Registry</span>

<span data-i18n="docsPage.gettingStarted.ociNoSetup">No repository setup is needed for OCI. You can install charts directly:</span>

```bash
helm install my-release oci://ghcr.io/helmforgedev/helm/redis
```

<span data-i18n="docsPage.gettingStarted.swap">You can swap `redis` for `komga`, `mosquitto`, `vaultwarden`, or any other chart published in the repository.</span>

## <span data-i18n="docsPage.gettingStarted.installFirst">Install Your First Chart</span>

<span data-i18n="docsPage.gettingStarted.redisExample">Let's install Redis as an example:</span>

### <span data-i18n="docsPage.gettingStarted.usingHttps">Using the HTTPS repository</span>

```bash
helm install my-redis helmforge/redis
```

### <span data-i18n="docsPage.gettingStarted.usingOci">Using the OCI registry</span>

```bash
helm install my-redis oci://ghcr.io/helmforgedev/helm/redis
```

### <span data-i18n="docsPage.gettingStarted.verifyInstall">Verify the installation</span>

```bash
kubectl get pods -l app.kubernetes.io/instance=my-redis
```

## <span data-i18n="docsPage.gettingStarted.customizing">Customizing Values</span>

<span data-i18n="docsPage.gettingStarted.defaults">Every chart ships with sensible defaults, but you can override any value:</span>

```bash
# See all available values
helm show values helmforge/redis

# Install with custom values
helm install my-redis helmforge/redis \
  --set architecture=sentinel \
  --set replicaCount=3
```

<span data-i18n="docsPage.gettingStarted.valuesFile">Or use a values file:</span>

```yaml
# my-values.yaml
architecture: sentinel
replicaCount: 3

metrics:
  enabled: true
  serviceMonitor:
    enabled: true
```

```bash
helm install my-redis helmforge/redis -f my-values.yaml
```
