# Wallabag

Deploy [Wallabag](https://wallabag.org/) on Kubernetes — a self-hosted read-it-later application for saving and organizing web articles with full-text search, tagging, and offline reading.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.auto.wallabag.b0">
    Article archiving — save articles from the web with full-text extraction
  </span>
- <span data-i18n="docsCharts.common.feature.postgresqlBackendBundledOrExternal">
    **PostgreSQL backend** — bundled subchart or external database
  </span>
- <span data-i18n="docsCharts.auto.wallabag.b1">Optional Redis — session caching and performance optimization</span>
- <span data-i18n="docsCharts.common.feature.s3BackupPostgresql">
    **S3 backup** — scheduled PostgreSQL backup to S3-compatible storage
  </span>
- <span data-i18n="docsCharts.common.feature.ingressCertManager">
    **Ingress support** — TLS via cert-manager with configurable ingress class
  </span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

```bash
helm install wallabag oci://ghcr.io/helmforgedev/helm/wallabag
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
postgresql:
enabled: true
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: wallabag.example.com
paths:
  - path: /
pathType: Prefix
```
