# ArchiveBox

Deploy [ArchiveBox](https://archivebox.io/) on Kubernetes ‚Äî a self-hosted web archiving platform that captures websites in multiple formats (HTML, PDF, PNG, WARC, media) using Chromium headless rendering.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>
- <span data-i18n="docsCharts.auto.archivebox.b0">Multi-format archiving ‚Äî HTML, PDF, screenshot, WARC, media extraction, git clone</span>
- <span data-i18n="docsCharts.auto.archivebox.b1">Chromium headless ‚Äî full browser rendering with /dev/shm memory-backed tmpfs</span>
- <span data-i18n="docsCharts.common.feature.sqliteDatabaseEmbedded">**SQLite database** ‚Äî embedded storage, no external database needed</span>
- <span data-i18n="docsCharts.auto.archivebox.b2">Persistent storage ‚Äî PVC for archived content and database</span>
- <span data-i18n="docsCharts.auto.archivebox.b3">Admin credentials ‚Äî managed via Kubernetes Secret with auto-generation</span>
- <span data-i18n="docsCharts.common.feature.ingressCertManager">**Ingress support** ‚Äî TLS via cert-manager with configurable ingress class</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install archivebox helmforge/archivebox -f values.yaml
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install archivebox oci://ghcr.io/helmforgedev/helm/archivebox -f values.yaml
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
archivebox:
adminUsername: admin
adminPassword: 'my-secure-password'
persistence:
enabled: true
size: 100Gi
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: archive.example.com
paths:
  - path: /
pathType: Prefix
```

## <span data-i18n="docsCharts.heading.operationalNotes">Operational Notes</span>
- single instance only ‚Äî SQLite is single-writer, no horizontal scaling- storage-heavy deployment ‚Äî plan 50‚Äì100GB+ PVC depending on archive volume- requires minimum 2Gi RAM for Chromium headless rendering- `/dev/shm` tmpfs is mounted automatically for Chromium stability
