# Authelia

Deploy [Authelia](https://www.authelia.com) on Kubernetes using the official [authelia/authelia](https://hub.docker.com/r/authelia/authelia) Docker image. Authentication and authorization server with single sign-on, multi-factor authentication, and OpenID Connect identity provider.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.auto.authelia.b0">Forward auth middleware — Traefik, nginx, Caddy, Envoy</span>
- <span data-i18n="docsCharts.auto.authelia.b1">Multi-factor authentication — TOTP, WebAuthn/FIDO2, Duo</span>
- <span data-i18n="docsCharts.auto.authelia.b2">OpenID Connect — certified identity provider for SSO</span>
- <span data-i18n="docsCharts.auto.authelia.b3">File or LDAP authentication backends</span>
- <span data-i18n="docsCharts.auto.authelia.b4">SQLite, PostgreSQL, or MySQL storage backends</span>
- <span data-i18n="docsCharts.auto.authelia.b5">Redis session storage for stateless HA deployments</span>
- <span data-i18n="docsCharts.auto.authelia.b6">Access control rules — domain, user, group, and network policies</span>
- <span data-i18n="docsCharts.auto.authelia.b7">Brute force protection — configurable regulation</span>
- <span data-i18n="docsCharts.auto.authelia.b8">Prometheus metrics with optional ServiceMonitor</span>
- <span data-i18n="docsCharts.auto.authelia.b9">S3 backup — SQLite tar, pg_dump, or mysqldump with S3 upload</span>
- <span data-i18n="docsCharts.auto.authelia.b10">Ingress support — TLS with cert-manager</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install authelia helmforge/authelia -f values.yaml
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install authelia oci://ghcr.io/helmforgedev/helm/authelia -f values.yaml
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span> (SQLite)

```yaml
secrets:
jwtSecret: 'your-64-char-random-string'
sessionSecret: 'your-64-char-random-string'
storageEncryptionKey: 'your-20-plus-char-string'
config:
session:
  cookies:
    - domain: example.com
authelia_url: 'https://auth.example.com'
access_control:
  default_policy:
one_factorusersDatabase:
  users:
admin:
  displayname: 'Admin'
email: 'admin@example.com'
password: '$argon2id$v=19$m=65536,t=3,p=4$...'
groups:
  - admins
```

After deploying, access via `kubectl port-forward svc/<release>-authelia 9091:80`.

## <span data-i18n="docsCharts.heading.postgresqlRedisMode">PostgreSQL + Redis Mode</span>

```yaml
database:
type: postgres
postgresql:
enabled: true
auth:
  password: 'change-me'
redis:
enabled: true
auth:
  password: 'change-me'
```

## <span data-i18n="docsCharts.heading.externalDatabase">External Database</span>

```yaml
database:
type: postgres
external:
  host: postgres.example.com
name: authelia
username: authelia
existingSecret: authelia-db-credentials
postgresql:
enabled: false
```
