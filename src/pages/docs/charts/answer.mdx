# Apache Answer

Deploy Apache Answer on Kubernetes using the official [apache/answer](https://hub.docker.com/r/apache/answer) Docker image. Supports SQLite (default), PostgreSQL, MySQL, automatic setup, and scheduled backups.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.auto.answer.b0">SQLite by Default — zero database configuration needed</span>
- <span data-i18n="docsCharts.common.feature.postgresqlSubchartHelmforge">
    **PostgreSQL Subchart** — bundled via HelmForge dependency
  </span>
- <span data-i18n="docsCharts.common.feature.mysqlSubchartHelmforge">
    **MySQL Subchart** — bundled via HelmForge dependency
  </span>
- <span data-i18n="docsCharts.common.feature.externalDatabasePostgresqlMysql">
    **External Database** — connect to existing PostgreSQL or MySQL
  </span>
- <span data-i18n="docsCharts.auto.answer.b1">Auto-Install — unattended setup via environment variables</span>
- <span data-i18n="docsCharts.auto.answer.b2">Scheduled Backups — database-aware CronJob with S3 upload</span>
- <span data-i18n="docsCharts.common.feature.ingressTlsCertManager">**Ingress Support** — TLS with cert-manager</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install answer helmforge/answer
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install answer oci://ghcr.io/helmforgedev/helm/answer
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span> (SQLite)

```yaml
# values.yaml
answer:
  siteName: 'My Q&A'
  admin:
    name: admin
    password: 'change-me'
persistence:
  enabled: true
  size: 5Gi
```

## <span data-i18n="docsCharts.heading.postgresqlExample">PostgreSQL Example</span>

```yaml
postgresql:
  enabled: true
  auth:
    database: answer
    username: answer
    password: 'strong-password'
ingress:
  enabled: true
  ingressClassName: traefik
  hosts:
    - host: qa.example.com
      paths:
        - path: /
          pathType: Prefix
  tls:
    - secretName: answer-tls
      hosts:
        - qa.example.com
```

## <span data-i18n="docsCharts.heading.externalDatabase">External Database</span> Example

```yaml
database:
  external:
    vendor: postgres
    host: db.example.com
    name: answer
    username: answer
    existingSecret: answer-db-credentials
```
