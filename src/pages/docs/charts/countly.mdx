# Countly

<span data-i18n="docsCharts.countly.intro.before">Deploy</span> [Countly](https://count.ly/)
<span data-i18n="docsCharts.countly.intro.after">
  on Kubernetes — a product analytics platform with event tracking, crash reporting, push notifications, A/B testing,
  and 41+ plugins.
</span>

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.countly.feature.fullAnalytics">
    Full analytics stack — event tracking, crash reporting, push notifications, A/B testing
  </span>
- <span data-i18n="docsCharts.countly.feature.mongodbBackend">
    MongoDB backend — bundled subchart or external connection
  </span>
- <span data-i18n="docsCharts.countly.feature.dualPorts">
    Dual ports — separate API (3001) and Dashboard (6001) endpoints
  </span>
- <span data-i18n="docsCharts.countly.feature.pluginSystem">
    Plugin system — configurable plugin list for feature selection
  </span>
- <span data-i18n="docsCharts.countly.feature.scalableWorkers">
    Scalable workers — configurable API worker processes
  </span>
- <span data-i18n="docsCharts.countly.feature.ingressSupport">
    Ingress support — TLS via cert-manager with configurable ingress class
  </span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install countly helmforge/countly -f values.yaml
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install countly oci://ghcr.io/helmforgedev/helm/countly -f values.yaml
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
countly:
  apiWorkers: 4

mongodb:
  enabled: true

ingress:
  enabled: true
  ingressClassName: traefik
  hosts:
    - host: countly.example.com
      paths:
        - path: /
          pathType: Prefix
```

## <span data-i18n="docsCharts.heading.externalDatabaseExample">External Database Example</span>

```yaml
mongodb:
  enabled: false

externalMongodb:
  enabled: true
  uri: 'mongodb://user:pass@mongo.example.internal:27017/countly'
```

## <span data-i18n="docsCharts.heading.operationalNotes">Operational Notes</span>

- <span data-i18n="docsCharts.countly.note.mongodbOnly">MongoDB only — no PostgreSQL support</span>
- <span data-i18n="docsCharts.countly.note.singleInstance">Single instance per MongoDB database</span>
- <span data-i18n="docsCharts.countly.note.sameContainer">
    Dashboard and API run in the same container on separate ports
  </span>
