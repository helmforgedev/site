# Vaultwarden

<span data-i18n="docsCharts.vaultwarden.intro">
  Self-hosted Bitwarden-compatible password manager. Vaultwarden is a lightweight, Rust-based implementation of the
  Bitwarden server API, compatible with all official Bitwarden clients.
</span>

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.vaultwarden.feature.compatible">
    <strong>Bitwarden compatible</strong> — Works with all official Bitwarden apps and browser extensions
  </span>
- <span data-i18n="docsCharts.vaultwarden.feature.lightweight">
    <strong>Lightweight</strong> — Minimal resource footprint compared to the official Bitwarden server
  </span>
- <span data-i18n="docsCharts.vaultwarden.feature.database">
    <strong>Database backends</strong> — SQLite, PostgreSQL, or MySQL
  </span>
- <span data-i18n="docsCharts.vaultwarden.feature.admin">
    <strong>Admin panel</strong> — Optional web-based administration interface
  </span>
- <span data-i18n="docsCharts.vaultwarden.feature.ingress">
    <strong>Ingress support</strong> — Configurable with `ingressClassName` and TLS
  </span>
- <span data-i18n="docsCharts.vaultwarden.feature.security">
    <strong>Security hardening</strong> — Network policies, non-root containers, pod security
  </span>
- <span data-i18n="docsCharts.vaultwarden.feature.storage">
    <strong>Persistent storage</strong> — Data and attachment persistence
  </span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install my-vault helmforge/vaultwarden
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install my-vault oci://ghcr.io/helmforgedev/helm/vaultwarden
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
# values.yaml
domain: 'https://vault.example.com'
admin:
  enabled: true
  token: 'my-admin-token'
ingress:
  enabled: true
  ingressClassName: traefik
  hosts:
    - host: vault.example.com
      paths:
        - path: /
          pathType: Prefix
  tls:
    - secretName: vaultwarden-tls
      hosts:
        - vault.example.com
persistence:
  enabled: true
  size: 10Gi
```

## <span data-i18n="docsCharts.heading.externalDatabase">External Database</span> Example

```yaml
domain: 'https://vault.example.com'
database:
  type: postgresql
  host: postgresql.database.svc
  port: 5432
  name: vaultwarden
  username: vaultwarden
  password: 'db-password'
persistence:
  enabled: true
  size: 5Gi
ingress:
  enabled: true
  ingressClassName: traefik
  hosts:
    - host: vault.example.com
      paths:
        - path: /
          pathType: Prefix
```
