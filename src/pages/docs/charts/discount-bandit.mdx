# Discount Bandit

Deploy Discount Bandit on Kubernetes using the official [cybrarist/discount-bandit](https://hub.docker.com/r/cybrarist/discount-bandit) container image. Discount Bandit is a self-hosted price tracker and deal aggregator that monitors prices across Amazon, eBay, AliExpress, and other stores.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.auto.discount_bandit.b0">
    Multi-Store Tracking — monitor prices on Amazon, eBay, AliExpress, and more
  </span>
- <span data-i18n="docsCharts.auto.discount_bandit.b1">
    SQLite Persistence — lightweight database with no external dependencies
  </span>
- <span data-i18n="docsCharts.auto.discount_bandit.b2">
    Auto-Generated App Key — Laravel application key generated automatically
  </span>
- <span data-i18n="docsCharts.auto.discount_bandit.b3">
    Price History — track price changes over time with historical data
  </span>
- <span data-i18n="docsCharts.auto.discount_bandit.b4">
    Deal Alerts — get notified when prices drop below your target
  </span>
- <span data-i18n="docsCharts.auto.discount_bandit.b5">
    Web Interface — manage tracked products through a clean dashboard
  </span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install discount-bandit helmforge/discount-bandit
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install discount-bandit oci://ghcr.io/helmforgedev/helm/discount-bandit
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
# values.yaml
discountBandit:
appUrl: 'https://deals.example.com'
persistence:
enabled: true
size: 5Gi
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: deals.example.com
paths:
  - path: /
pathType: Prefix
```
