# Apache Guacamole

<span data-i18n="docsCharts.guacamole.intro">
  Deploy Apache Guacamole on Kubernetes using the official `guacamole/guacamole` and `guacamole/guacd` Docker images. A
  clientless remote desktop gateway supporting RDP, VNC, SSH, and telnet with OIDC/SAML SSO.
</span>

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.guacamole.feature.guacdSidecar">
    guacd Sidecar — protocol daemon runs alongside the web app
  </span>
- <span data-i18n="docsCharts.guacamole.feature.postgresqlSubchart">
    PostgreSQL Subchart — bundled via HelmForge dependency (default)
  </span>
- <span data-i18n="docsCharts.guacamole.feature.mysqlSubchart">MySQL Subchart — bundled via HelmForge dependency</span>
- <span data-i18n="docsCharts.guacamole.feature.externalDatabase">
    External Database — connect to existing PostgreSQL or MySQL
  </span>
- <span data-i18n="docsCharts.guacamole.feature.databaseInitJob">
    Database Init Job — automatic schema initialization on install
  </span>
- <span data-i18n="docsCharts.guacamole.feature.openidConnect">
    OpenID Connect — SSO with Keycloak, Okta, Azure AD, and others
  </span>
- <span data-i18n="docsCharts.guacamole.feature.saml">SAML — SSO with any SAML 2.0 identity provider</span>
- <span data-i18n="docsCharts.guacamole.feature.totp">TOTP — two-factor authentication</span>
- <span data-i18n="docsCharts.guacamole.feature.scheduledBackups">
    Scheduled Backups — pg_dump or mysqldump with S3 upload
  </span>
- <span data-i18n="docsCharts.guacamole.feature.ingressSupport">Ingress Support — TLS with cert-manager</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install guacamole helmforge/guacamole -f values.yaml
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install guacamole oci://ghcr.io/helmforgedev/helm/guacamole -f values.yaml
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span> (PostgreSQL)

```yaml
# values.yaml
postgresql:
  auth:
    password: 'change-me'
```

<span data-i18n="docsCharts.guacamole.defaultCredentials">
  Default credentials: `guacadmin` / `guacadmin` — change immediately after first login.
</span>

## <span data-i18n="docsCharts.heading.oidcWithKeycloak">OIDC with Keycloak</span>

```yaml
oidc:
  enabled: true
  authorizationEndpoint: https://keycloak.example.com/realms/master/protocol/openid-connect/auth
  jwksEndpoint: https://keycloak.example.com/realms/master/protocol/openid-connect/certs
  issuer: https://keycloak.example.com/realms/master
  clientId: guacamole

ingress:
  enabled: true
  ingressClassName: traefik
  hosts:
    - host: guacamole.example.com
      paths:
        - path: /
          pathType: Prefix
  tls:
    - secretName: guacamole-tls
      hosts:
        - guacamole.example.com
```

<span data-i18n="docsCharts.guacamole.redirectUri">
  The `redirectUri` is auto-detected from the ingress configuration.
</span>

## <span data-i18n="docsCharts.heading.externalDatabase">External Database</span>

```yaml
postgresql:
  enabled: false

database:
  type: postgresql
  external:
    host: db.example.com
    name: guacamole_db
    username: guacamole_user
    existingSecret: guacamole-db-credentials
```
