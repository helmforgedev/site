# phpMyAdmin

Helm chart for deploying [phpMyAdmin](https://www.phpmyadmin.net/) on Kubernetes using the official [`phpmyadmin/phpmyadmin`](https://hub.docker.com/r/phpmyadmin/phpmyadmin) Docker image.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.auto.phpmyadmin.b0">Official phpMyAdmin image based on `phpmyadmin/phpmyadmin`</span>
- <span data-i18n="docsCharts.auto.phpmyadmin.b1">
    Single or multi-server connect to one MySQL host or multiple servers
  </span>
- <span data-i18n="docsCharts.auto.phpmyadmin.b2">
    Auto-login optional automatic authentication via credentials or existing secret
  </span>
- <span data-i18n="docsCharts.auto.phpmyadmin.b3">Custom configuration mount `config.user.inc.php` via ConfigMap</span>
- <span data-i18n="docsCharts.auto.phpmyadmin.b4">Upload limit configurable max upload size for SQL imports</span>
- <span data-i18n="docsCharts.auto.phpmyadmin.b5">Stateless no persistent storage needed, scales horizontally</span>
- <span data-i18n="docsCharts.common.feature.ingressConfigurableTls">
    **Ingress support** configurable ingress with TLS
  </span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**HTTPS Repository**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install phpmyadmin helmforge/phpmyadmin --set phpmyadmin.host=mysql.default.svc
```

**OCI Registry**

```bash
helm install phpmyadmin oci://ghcr.io/helmforgedev/helm/phpmyadmin --set phpmyadmin.host=mysql.default.svc
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
phpmy
admin:
host: mysql.default.svc.cluster.local
port: 3306
uploadLimit: '128M'
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: pma.example.com
paths:
  - path: /
pathType: Prefix
tls:
  - secretName: phpmyadmin-tls
hosts:
  - pma.example.com
```
