# Minecraft

Deploy Minecraft Java Edition servers on Kubernetes using the [itzg/minecraft-server](https://docker-minecraft-server.readthedocs.io/) container image. Supports multiple server types, cross-play, modding, monitoring, and scheduled backups.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.auto.minecraft.b0">
    Multiple Server Types — Vanilla, Paper, Spigot, Forge, Fabric, Quilt
  </span>
- <span data-i18n="docsCharts.auto.minecraft.b1">
    GeyserMC Cross-Play — Allow Bedrock clients (mobile, console) to join Java servers
  </span>
- <span data-i18n="docsCharts.auto.minecraft.b2">Authentication — Online/offline mode, whitelist, operators</span>
- <span data-i18n="docsCharts.auto.minecraft.b3">RCON — Remote console with auto-generated password</span>
- <span data-i18n="docsCharts.auto.minecraft.b4">
    Scheduled Backups — CronJob-based backup to S3-compatible storage
  </span>
- <span data-i18n="docsCharts.auto.minecraft.b5">Prometheus Monitoring — mc-monitor sidecar with ServiceMonitor</span>
- <span data-i18n="docsCharts.auto.minecraft.b6">Mod/Plugin Support — Modrinth, CurseForge, Spiget, direct URLs</span>
- <span data-i18n="docsCharts.auto.minecraft.b7">Optimized JVM — Aikar's GC flags for production workloads</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install minecraft helmforge/minecraft --set server.eula=true
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install minecraft oci://ghcr.io/helmforgedev/helm/minecraft --set server.eula=true
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
# values.yaml
server:
eula: true
type: PAPER
motd: 'My Minecraft Server'
maxPlayers: 20jvm:
memory: 2G
useAikarFlags: true
auth:
onlineMode: true
whitelist: 'player1,player2'
ops: 'player1'
persistence:
enabled: true
size: 20Gi
```

## <span data-i18n="docsCharts.heading.crossPlayExample">Cross-Play Example</span>

```yaml
server:
eula: true
type:
  PAPERgeyser:
enabled: true
port: 19132mods:
modrinthProjects: 'geyser,floodgate'
```
