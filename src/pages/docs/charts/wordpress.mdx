# WordPress

<span data-i18n="docsCharts.wordpress.intro.before">Deploy WordPress on Kubernetes using the official</span> [wordpress](https://hub.docker.com/_/wordpress) <span data-i18n="docsCharts.wordpress.intro.after">Docker image (Apache variant). Supports MySQL subchart or external database, scheduled backups, and monitoring.</span>

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>
- <span data-i18n="docsCharts.wordpress.feature.officialImage"><strong>Official WordPress Image</strong> — Apache variant with PHP</span>
- <span data-i18n="docsCharts.wordpress.feature.mysqlSubchart"><strong>MySQL Subchart</strong> — Bundled MySQL via HelmForge dependency</span>
- <span data-i18n="docsCharts.wordpress.feature.externalDb"><strong>External Database</strong> — Connect to existing MySQL/MariaDB</span>
- <span data-i18n="docsCharts.wordpress.feature.backups"><strong>Scheduled Backups</strong> — mysqldump + wp-content archive to S3</span>
- <span data-i18n="docsCharts.wordpress.feature.ingress"><strong>Ingress Support</strong> — TLS with cert-manager</span>
- <span data-i18n="docsCharts.wordpress.feature.phpConfig"><strong>PHP Configuration</strong> — Custom php.ini via ConfigMap</span>
- <span data-i18n="docsCharts.wordpress.feature.metrics"><strong>Prometheus Metrics</strong> — Apache exporter sidecar</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install wordpress helmforge/wordpress
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install wordpress oci://ghcr.io/helmforgedev/helm/wordpress
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
# values.yaml
wordpress:
  siteTitle: 'My Blog'
  adminUser: admin
  adminPassword: 'change-me'
mysql:
  enabled: true
  auth:
    password: 'db-password'
persistence:
  enabled: true
  size: 5Gi
ingress:
  enabled: true
  ingressClassName: traefik
  hosts:
    - host: blog.example.com
      paths:
        - path: /
          pathType: Prefix
  tls:
    - secretName: wordpress-tls
      hosts:
        - blog.example.com
```

## <span data-i18n="docsCharts.heading.externalDatabase">External Database</span> Example

```yaml
mysql:
  enabled: false
database:
  external:
    host: db.example.com
    name: wordpress
    username: wordpress
    existingSecret: wordpress-db
```
