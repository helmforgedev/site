# Dolibarr

Helm chart for deploying [Dolibarr ERP/CRM](https://www.dolibarr.org/) on Kubernetes using the official [`dolibarr/dolibarr`](https://hub.docker.com/r/dolibarr/dolibarr) Docker image.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>
- <span data-i18n="docsCharts.auto.dolibarr.b0">Official Dolibarr image based on `dolibarr/dolibarr`</span>
- <span data-i18n="docsCharts.auto.dolibarr.b1">MySQL subchart bundled HelmForge MySQL dependency</span>
- <span data-i18n="docsCharts.auto.dolibarr.b2">External MySQL/MariaDB connect to a managed database</span>
- <span data-i18n="docsCharts.auto.dolibarr.b3">Auto installation unattended setup via `DOLI_*` environment variables</span>
- <span data-i18n="docsCharts.auto.dolibarr.b4">Persistent storage separate PVCs for documents and custom modules</span>
- <span data-i18n="docsCharts.common.feature.ingressConfigurableTls">**Ingress support** configurable ingress with TLS</span>
- <span data-i18n="docsCharts.auto.dolibarr.b5">Secret preservation admin, runtime, and database secrets preserved across upgrades</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**HTTPS Repository**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install dolibarr helmforge/dolibarr
```

**OCI Registry**

```bash
helm install dolibarr oci://ghcr.io/helmforgedev/helm/dolibarr
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
dolibarr:
  siteUrl: 'https://erp.example.com'
  companyName: 'Example Corp'
  companyCountryCode: 'US'
  enableModules: 'societe,produit,service,propal,commande,facture'
mysql:
  enabled: true
  auth:
    password: 'change-me'
ingress:
  enabled: true
  ingressClassName: traefik
  hosts:
    - host: erp.example.com
      paths:
        - path: /
          pathType: Prefix
  tls:
    - secretName: dolibarr-tls
      hosts:
        - erp.example.com
```
