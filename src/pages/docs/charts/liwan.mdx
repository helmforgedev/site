# Liwan

Deploy [Liwan](https://github.com/explodingcamera/liwan) on Kubernetes — an ultra-lightweight, privacy-first web analytics tool with embedded DuckDB storage and minimal resource requirements.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.auto.liwan.b0">Minimal footprint — single binary with embedded DuckDB</span>
- <span data-i18n="docsCharts.auto.liwan.b1">Privacy-first — no cookies, no tracking, GDPR-compliant</span>
- <span data-i18n="docsCharts.auto.liwan.b2">Persistent storage — PVC-backed data directory</span>
- <span data-i18n="docsCharts.common.feature.ingressCertManager">
    **Ingress support** — TLS via cert-manager with configurable ingress class
  </span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

```bash
helm install liwan oci://ghcr.io/helmforgedev/helm/liwan
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
persistence:
enabled: true
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: analytics.example.com
paths:
  - path: /
pathType: Prefix
```
