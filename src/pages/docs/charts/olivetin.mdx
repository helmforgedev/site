# OliveTin

Deploy OliveTin on Kubernetes using the official [jamesread/olivetin](https://hub.docker.com/r/jamesread/olivetin) container image. OliveTin provides a clean web interface for running predefined shell commands, making it easy to give safe access to common system actions.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.auto.olivetin.b0">
    ConfigMap-Based Actions — define shell commands in YAML, mounted as a ConfigMap
  </span>
- <span data-i18n="docsCharts.auto.olivetin.b1">
    Web UI — clean, responsive browser interface for triggering actions
  </span>
- <span data-i18n="docsCharts.auto.olivetin.b2">
    Prometheus Metrics — optional metrics scraping with ServiceMonitor support
  </span>
- <span data-i18n="docsCharts.auto.olivetin.b3">Customizable — action titles, icons, and confirmation prompts</span>
- <span data-i18n="docsCharts.auto.olivetin.b4">
    Lightweight — minimal resource footprint with no external dependencies
  </span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install olivetin helmforge/olivetin
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install olivetin oci://ghcr.io/helmforgedev/helm/olivetin
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
# values.yaml
config: |
actions:
  - title: "Ping Google"
shell: "ping -c 1 google.com"
  - title: "Show Disk Usage"
shell: "df -h"
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: olivetin.example.com
paths:
  - path: /
pathType: Prefix
```
