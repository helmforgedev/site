# n8n

<span data-i18n="docsCharts.n8n.intro.before">Deploy n8n on Kubernetes using the official</span>
[n8nio/n8n](https://hub.docker.com/r/n8nio/n8n)
<span data-i18n="docsCharts.n8n.intro.after">
  Docker image. Supports SQLite (default), PostgreSQL, MySQL, Redis-backed queue mode with dedicated workers, and
  scheduled backups.
</span>

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.n8n.feature.sqlite">
    <strong>SQLite by Default</strong> — zero database configuration needed
  </span>
- <span data-i18n="docsCharts.n8n.feature.postgresSubchart">
    <strong>PostgreSQL Subchart</strong> — bundled via HelmForge dependency
  </span>
- <span data-i18n="docsCharts.n8n.feature.mysqlSubchart">
    <strong>MySQL Subchart</strong> — bundled via HelmForge dependency
  </span>
- <span data-i18n="docsCharts.n8n.feature.externalDb">
    <strong>External Database</strong> — connect to existing PostgreSQL or MySQL
  </span>
- <span data-i18n="docsCharts.n8n.feature.queue">
    <strong>Queue Mode</strong> — Redis-backed horizontal scaling with worker Deployments
  </span>
- <span data-i18n="docsCharts.n8n.feature.encryptionKey">
    <strong>Encryption Key</strong> — auto-generated and preserved across upgrades
  </span>
- <span data-i18n="docsCharts.n8n.feature.backups">
    <strong>Scheduled Backups</strong> — database-aware CronJob with S3 upload
  </span>
- <span data-i18n="docsCharts.n8n.feature.ingress">
    <strong>Ingress Support</strong> — TLS with cert-manager
  </span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install n8n helmforge/n8n
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install n8n oci://ghcr.io/helmforgedev/helm/n8n
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span> (SQLite)

```yaml
# values.yaml
persistence:
  enabled: true
  size: 5Gi
ingress:
  enabled: true
  ingressClassName: traefik
  hosts:
    - host: n8n.example.com
      paths:
        - path: /
          pathType: Prefix
  tls:
    - secretName: n8n-tls
      hosts:
        - n8n.example.com
```

## <span data-i18n="docsCharts.heading.postgresqlExample">PostgreSQL Example</span>

```yaml
postgresql:
  enabled: true
  auth:
    database: n8n
    username: n8n
    password: 'strong-password'
n8n:
  webhookUrl: 'https://n8n.example.com'
```

## <span data-i18n="docsCharts.heading.queueModeExample">Queue Mode Example</span>

```yaml
queue:
  enabled: true
  workers: 3
  concurrency: 10
redis:
  enabled: true
  auth:
    password: 'redis-secret'
```

## <span data-i18n="docsCharts.heading.externalDatabase">External Database</span> Example

```yaml
database:
  external:
    vendor: postgres
    host: db.example.com
    name: n8n
    username: n8n
    existingSecret: n8n-db-credentials
```
