# changedetection.io

Deploy changedetection.io on Kubernetes using the official [dgtlmoon/changedetection.io](https://github.com/dgtlmoon/changedetection.io) container image. Monitor websites for changes with support for CSS/XPath selectors, visual comparisons, and notifications via email, webhooks, and more.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>
- <span data-i18n="docsCharts.auto.changedetection.b0">Website Monitoring — detect changes on any web page with configurable check intervals</span>
- <span data-i18n="docsCharts.auto.changedetection.b1">CSS/XPath Selectors — target specific page elements for precise change detection</span>
- <span data-i18n="docsCharts.auto.changedetection.b2">Browser Sidecar — optional Playwright/Chromium sidecar for JavaScript-rendered pages</span>
- <span data-i18n="docsCharts.auto.changedetection.b3">Notifications — email, webhook, Slack, Telegram, and many more notification channels</span>
- <span data-i18n="docsCharts.auto.changedetection.b4">REST API — programmatic access to watches and change history</span>
- <span data-i18n="docsCharts.auto.changedetection.b5">Persistence — SQLite database and page snapshots stored on persistent volume</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install changedetection helmforge/changedetection
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install changedetection oci://ghcr.io/helmforgedev/helm/changedetection
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
# values.yaml
changedetection:
baseUrl: 'https://changes.example.com'browser:
enabled: true
persistence:
enabled: true
size: 10Gi
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: changes.example.com
paths:
  - path: /
pathType: Prefix
```
