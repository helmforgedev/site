# Castopod

Deploy Castopod on Kubernetes using the official [castopod/castopod](https://hub.docker.com/r/castopod/castopod) container image. Castopod is an open-source podcast hosting platform with built-in analytics, Fediverse integration, and RSS feed generation.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>
- <span data-i18n="docsCharts.auto.castopod.b0">MariaDB Backend — bundled MariaDB subchart or external database support</span>
- <span data-i18n="docsCharts.auto.castopod.b1">Redis Cache — optional Redis subchart for improved performance</span>
- <span data-i18n="docsCharts.auto.castopod.b2">RSS Feed — automatic podcast RSS feed generation</span>
- <span data-i18n="docsCharts.auto.castopod.b3">Analytics — built-in podcast analytics with hashed listener tracking</span>
- <span data-i18n="docsCharts.auto.castopod.b4">Fediverse Integration — ActivityPub support for social interactions</span>
- <span data-i18n="docsCharts.auto.castopod.b5">Persistence — persistent storage for uploads, cache, and session data</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install castopod helmforge/castopod
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install castopod oci://ghcr.io/helmforgedev/helm/castopod
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
# values.yaml
castopod:
baseURL: 'https://podcast.example.com'mariadb:
enabled: true
redis:
enabled: true
persistence:
enabled: true
size: 10Gi
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: podcast.example.com
paths:
  - path: /
pathType: Prefix
```
