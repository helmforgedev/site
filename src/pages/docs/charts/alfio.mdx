# alf.io

Deploy alf.io on Kubernetes using the official [alfio/alf.io](https://hub.docker.com/r/alfio/alf.io) container image. alf.io is an open-source event management and ticketing system built with Spring Boot, supporting event creation, ticket sales, check-in, and attendee management.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>
- <span data-i18n="docsCharts.common.feature.postgresqlBackendBundledOrExternalSupport">**PostgreSQL Backend** — bundled PostgreSQL subchart or external database support</span>
- <span data-i18n="docsCharts.auto.alfio.b0">Spring Boot — production-ready Java application with configurable profiles</span>
- <span data-i18n="docsCharts.auto.alfio.b1">Event Management — create and manage events, ticket categories, and promotions</span>
- <span data-i18n="docsCharts.auto.alfio.b2">Ticket Sales — online ticket purchasing with payment gateway integration</span>
- <span data-i18n="docsCharts.auto.alfio.b3">Check-In — QR code-based attendee check-in system</span>
- <span data-i18n="docsCharts.common.feature.ingressSupportConfigurableTls">**Ingress Support** — configurable ingress with TLS</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install alfio helmforge/alfio
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install alfio oci://ghcr.io/helmforgedev/helm/alfio
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
# values.yaml
alfio:
baseUrl: 'https://tickets.example.com'
profiles: 'dev'
postgresql:
enabled: true
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: tickets.example.com
paths:
  - path: /
pathType: Prefix
```
