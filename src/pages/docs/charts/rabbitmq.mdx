# RabbitMQ

<span data-i18n="docsCharts.rabbitmq.intro">
  Production-ready RabbitMQ deployment with management UI and clustering support.
</span>

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.rabbitmq.feature.standaloneClustering">
    Standalone and clustering — Single instance or multi-node cluster
  </span>
- <span data-i18n="docsCharts.rabbitmq.feature.managementUi">
    Management UI — Built-in web management console with ingress support
  </span>
- <span data-i18n="docsCharts.rabbitmq.feature.pluginSupport">Plugin support — Configurable RabbitMQ plugins</span>
- <span data-i18n="docsCharts.rabbitmq.feature.metrics">Metrics — Prometheus exporter with ServiceMonitor</span>
- <span data-i18n="docsCharts.rabbitmq.feature.security">
    Security — Authentication, non-root containers, network policies, TLS
  </span>
- <span data-i18n="docsCharts.rabbitmq.feature.persistentStorage">
    Persistent storage — Message persistence across restarts
  </span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install my-rabbit helmforge/rabbitmq
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install my-rabbit oci://ghcr.io/helmforgedev/helm/rabbitmq
```

## <span data-i18n="docsCharts.heading.standaloneExample">Standalone Example</span>

```yaml
# values.yaml
auth:
  username: admin
  password: 'my-secret-password'

persistence:
  size: 10Gi

metrics:
  enabled: true
  serviceMonitor:
    enabled: true
```

## <span data-i18n="docsCharts.heading.clusterExample">Cluster Example</span>

```yaml
replicaCount: 3

auth:
  username: admin
  password: 'my-secret-password'
  erlangCookie: 'my-erlang-cookie'

persistence:
  size: 10Gi

ingress:
  enabled: true
  ingressClassName: traefik
  hosts:
    - host: rabbitmq.example.com
      paths:
        - path: /
          pathType: Prefix
  tls:
    - secretName: rabbitmq-tls
      hosts:
        - rabbitmq.example.com
```
