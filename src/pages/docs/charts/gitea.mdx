# Gitea

Helm chart for deploying [Gitea](https://gitea.io/) self-hosted Git service on Kubernetes using the official [`gitea/gitea`](https://hub.docker.com/r/gitea/gitea) rootless Docker image.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>
- <span data-i18n="docsCharts.auto.gitea.b0">Official rootless image based on `gitea/gitea:<version>-rootless` (UID 1000)</span>
- <span data-i18n="docsCharts.common.feature.databaseBackendsAutoDetection">**Database backends** SQLite3 (default), PostgreSQL, or MySQL with auto-detection</span>
- <span data-i18n="docsCharts.common.feature.postgresqlMysqlSubchartsOptional">**PostgreSQL and MySQL subcharts** optional bundled database deployments</span>
- <span data-i18n="docsCharts.auto.gitea.b1">HTTP + SSH services separate services for web UI and Git SSH access</span>
- <span data-i18n="docsCharts.auto.gitea.b2">Admin user creation optional post-install Job to bootstrap an admin account</span>
- <span data-i18n="docsCharts.common.feature.s3CompatibleBackupDatabaseAware">**S3-compatible backup** database-aware CronJob (SQLite tar, pg_dump, mysqldump)</span>
- <span data-i18n="docsCharts.common.feature.ingressConfigurableTls">**Ingress support** configurable ingress with TLS</span> for HTTP access

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**HTTPS Repository**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install gitea helmforge/gitea
```

**OCI Registry**

```bash
helm install gitea oci://ghcr.io/helmforgedev/helm/gitea
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
gitea:
rootUrl:
  https://git.example.com/
sshDomain: git.example.com
disableRegistration: true
admin:
username: gitea_admin
password: 'change-me-now'
email: admin@example.com
postgresql:
enabled: true
auth:
  database: gitea
username: gitea
password: 'db-password'
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: git.example.com
paths:
  - path: /
pathType: Prefix
tls:
  - secretName: gitea-tls
hosts:
  - git.example.com
backup:
enabled: true
schedule: '0 3 * * *'
s3:
  endpoint:
https://s3.amazonaws.com
bucket: my-backups
prefix: gitea
existingSecret: gitea-s3-credentials
```

## <span data-i18n="docsCharts.heading.databaseAutoDetection">Database Auto-Detection</span>

When `database.mode` is `auto` (default), the chart detects which database to use:1. If `database.external.host` is set → **external** database2. If `postgresql.enabled` is `true` → **PostgreSQL subchart**3. If `mysql.enabled` is `true` → **MySQL subchart**4. Otherwise → **SQLite3** (zero configuration)
