# Strava Statistics

Deploy [Statistics for Strava](https://github.com/robiningelbrecht/strava-statistics) on Kubernetes — a self-hosted fitness dashboard that visualizes your Strava activity data with detailed charts and statistics.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>
- <span data-i18n="docsCharts.auto.strava_statistics.b0">Strava OAuth — automated data sync from your Strava account</span>
- <span data-i18n="docsCharts.common.feature.sqliteDatabaseEmbedded">**SQLite database** — embedded storage, no external database needed</span>
- <span data-i18n="docsCharts.auto.strava_statistics.b1">Persistent storage — PVC for activity data and database</span>
- <span data-i18n="docsCharts.common.feature.ingressCertManager">**Ingress support** — TLS via cert-manager with configurable ingress class</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

```bash
helm install strava-statistics oci://ghcr.io/helmforgedev/helm/strava-statistics
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
stravaStatistics:
clientId: 'your-strava-client-id'
clientSecret: 'your-strava-client-secret'
refreshToken: 'your-strava-refresh-token'
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: strava.example.com
paths:
  - path: /
pathType: Prefix
```
