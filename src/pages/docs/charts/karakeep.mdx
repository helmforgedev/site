# Karakeep

Deploy Karakeep on Kubernetes using the official [karakeep-app/karakeep](https://github.com/karakeep-app/karakeep) container image. Karakeep is an AI-powered bookmark manager with full-text search via Meilisearch, automatic web page screenshots via Chromium, and intelligent tagging.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.auto.karakeep.b0">
    Meilisearch Sidecar — optional full-text search engine running alongside the app
  </span>
- <span data-i18n="docsCharts.auto.karakeep.b1">
    Chromium Sidecar — optional headless browser for web page screenshots and archiving
  </span>
- <span data-i18n="docsCharts.auto.karakeep.b2">AI Tagging — automatic bookmark categorization and tagging</span>
- <span data-i18n="docsCharts.auto.karakeep.b3">
    Full-Text Search — search across bookmarks, tags, and archived content
  </span>
- <span data-i18n="docsCharts.auto.karakeep.b4">
    Persistence — persistent storage for SQLite database and uploaded files
  </span>
- <span data-i18n="docsCharts.auto.karakeep.b5">Lightweight — single-pod deployment with optional sidecars</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install karakeep helmforge/karakeep
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install karakeep oci://ghcr.io/helmforgedev/helm/karakeep
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
# values.yaml
karakeep:
nextAuthUrl: 'https://karakeep.example.com'meilisearch:
enabled:
  truechromium:
enabled: true
persistence:
enabled: true
size: 10Gi
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: karakeep.example.com
paths:
  - path: /
pathType: Prefix
```
