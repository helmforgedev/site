# Ghost

Deploy Ghost on Kubernetes using the official [ghost](https://hub.docker.com/_/ghost) container image. Ghost is a powerful open-source publishing platform for blogs, newsletters, and memberships with a modern editor and headless CMS API.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>

- <span data-i18n="docsCharts.auto.ghost.b0">MySQL Backend — bundled MySQL subchart or external database support</span>
- <span data-i18n="docsCharts.auto.ghost.b1">
    Content Persistence — persistent volume for images, media, and uploaded files
  </span>
- <span data-i18n="docsCharts.auto.ghost.b2">
    S3 Backup — scheduled content backups to S3-compatible object storage
  </span>
- <span data-i18n="docsCharts.auto.ghost.b3">Headless CMS API — Content API and Admin API for headless usage</span>
- <span data-i18n="docsCharts.auto.ghost.b4">
    Memberships & Newsletters — built-in subscription and email newsletter support
  </span>
- <span data-i18n="docsCharts.common.feature.ingressSupportConfigurableTls">
    **Ingress Support** — configurable ingress with TLS
  </span>
  for the publishing frontend

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install ghost helmforge/ghost
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install ghost oci://ghcr.io/helmforgedev/helm/ghost
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
# values.yaml
ghost:
url: 'https://blog.example.com'mysql:
enabled: true
persistence:
enabled: true
size: 10Gi
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: blog.example.com
paths:
  - path: /
pathType: Prefix
```

## <span data-i18n="docsCharts.heading.keyValues">Key Values</span>

| <span data-i18n="docsCharts.table.key">Key</span> | <span data-i18n="docsCharts.table.default">Default</span> | <span data-i18n="docsCharts.table.description">Description</span>

| --------------------- | ----------- | ----------------------------------- |
| `ghost.url` | `""` | <span data-i18n="docsCharts.auto.ghost.t0">Public URL of the Ghost instance</span> |
| `mysql.enabled` | `true` | <span data-i18n="docsCharts.auto.ghost.t1">Enable bundled MySQL subchart</span> |
| `persistence.enabled` | `true` | <span data-i18n="docsCharts.auto.ghost.t2">Enable content persistence</span> |
| `persistence.size` | `10Gi` | <span data-i18n="docsCharts.common.pvcSize">PVC size</span> for content |
| `backup.enabled` | `false` | <span data-i18n="docsCharts.auto.ghost.t3">Enable scheduled S3 content backups</span> |
| `backup.schedule` | `0 3 * * *` | <span data-i18n="docsCharts.auto.ghost.t4">Cron schedule for backups</span> |
| `ingress.enabled` | `false` | <span data-i18n="docsCharts.common.enableIngress">Enable ingress</span> |
