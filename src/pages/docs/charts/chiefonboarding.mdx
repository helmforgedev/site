# ChiefOnboarding

Deploy ChiefOnboarding on Kubernetes using the official [chiefonboarding/chiefonboarding](https://hub.docker.com/r/chiefonboarding/chiefonboarding) container image. ChiefOnboarding is an employee onboarding automation platform with task sequences, integrations, and a clean web interface.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>
- <span data-i18n="docsCharts.common.feature.postgresqlBackendBundledOrExternalSupport">**PostgreSQL Backend** — bundled PostgreSQL subchart or external database support</span>
- <span data-i18n="docsCharts.auto.chiefonboarding.b0">Task Sequences — define onboarding workflows with ordered tasks and triggers</span>
- <span data-i18n="docsCharts.auto.chiefonboarding.b1">Auto-Generated Secret — Django secret key generated automatically if not provided</span>
- <span data-i18n="docsCharts.auto.chiefonboarding.b2">Integrations — connect with Slack, Google Workspace, and other services</span>
- <span data-i18n="docsCharts.auto.chiefonboarding.b3">Web Interface — admin dashboard for managing onboarding processes</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install chiefonboarding helmforge/chiefonboarding
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install chiefonboarding oci://ghcr.io/helmforgedev/helm/chiefonboarding
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
# values.yaml
chiefonboarding:
baseUrl: 'https://onboarding.example.com'
postgresql:
enabled: true
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: onboarding.example.com
paths:
  - path: /
pathType: Prefix
```
