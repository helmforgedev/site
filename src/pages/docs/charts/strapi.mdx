# Strapi

Deploy Strapi on Kubernetes as a headless CMS for APIs, websites, and custom admin experiences. The chart is designed for a prebuilt Strapi project image and supports SQLite, PostgreSQL, MySQL, persistent uploads, and scheduled backups.Set `ingress.ingressClassName` to the controller available in your cluster, such as `traefik`, `nginx`, or another supported ingress class.

## <span data-i18n="docsCharts.heading.keyFeatures">Key Features</span>
- <span data-i18n="docsCharts.auto.strapi.b0">Prebuilt project image — intended for your own Strapi application image</span>
- <span data-i18n="docsCharts.auto.strapi.b1">SQLite by default — simple initial deployment path</span>
- <span data-i18n="docsCharts.auto.strapi.b2">PostgreSQL subchart — bundled via HelmForge dependency</span>
- <span data-i18n="docsCharts.auto.strapi.b3">MySQL subchart — bundled via HelmForge dependency</span>
- <span data-i18n="docsCharts.auto.strapi.b4">External database — connect to existing PostgreSQL or MySQL</span>
- <span data-i18n="docsCharts.auto.strapi.b5">Application secrets — auto-generated app keys and token salts</span>
- <span data-i18n="docsCharts.auto.strapi.b6">Uploads persistence — PVC-backed media storage</span>
- <span data-i18n="docsCharts.auto.strapi.b7">Scheduled backups — SQLite archive or SQL dump workflows to S3</span>

## <span data-i18n="docsCharts.heading.installation">Installation</span>

**<span data-i18n="docsCharts.install.httpsRepo">HTTPS repository:</span>**

```bash
helm repo add helmforge https://repo.helmforge.dev
helm repo update
helm install strapi helmforge/strapi
```

**<span data-i18n="docsCharts.install.ociRegistry">OCI registry:</span>**

```bash
helm install strapi oci://ghcr.io/helmforgedev/helm/strapi
```

## <span data-i18n="docsCharts.heading.basicExample">Basic Example</span>

```yaml
image:
repository: ghcr.io/example/my-strapi
tag: '1.0.0'
persistence:
enabled: true
size: 5Gi
```

## <span data-i18n="docsCharts.heading.postgresqlExample">PostgreSQL Example</span>

```yaml
image:
repository: ghcr.io/example/my-strapi
tag: '1.0.0'
postgresql:
enabled: true
auth:
  database: strapi
username: strapi
password: 'strong-password'
ingress:
enabled: true
ingressClassName: traefik
hosts:
  - host: cms.example.com
paths:
  - path: /
pathType: Prefix
tls:
  - secretName: strapi-tls
hosts:
  - cms.example.com
```

## <span data-i18n="docsCharts.heading.externalDatabase">External Database</span> Example

```yaml
database:
mode: external
external:
  vendor: postgres
host: db.example.com
name: strapi
username: strapi
existingSecret: strapi-db-credentials
```
